# webletton

> a sample boilerplate for static web pages

##uses

 - coffeescript
 - coffeelint
 - jade
 - stylus with nib
 - uglify
 - connect server
 - livereload
 - watch and newer

#grunt comands

 - default
   - build
   - dev
 - build
   - clean public folder;
   - lint, compile and  uglify scripts;
   - compile jade and stylus;
 - dev
   - start server;
   - use livereload;
 - check
   - run only server;
